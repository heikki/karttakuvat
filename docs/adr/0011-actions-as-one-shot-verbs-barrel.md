# `@common/actions` as a one-shot verbs barrel

`@common/actions` exports the one-shot verbs Lit components reach for when they need to do something that isn't a signal write: modal openers (`showLightbox`, `showMetadata`, `showAlbumFiles`), MapApi forwarders (`fitToPhotos`, `openExternalMap`, `reloadAlbumGpx`), and `saveEdits`. Considered stripping it to true cross-store orchestrations only — modal openers would move to direct imports, MapApi forwarders would be replaced by a `getMapApi()` accessor — but the depth win didn't justify the loss of a single discoverable import path for component handlers. Multi-step orchestrations (e.g. the Reset button's filter + URL + viewState + map sequence) stay inlined at their call site instead of hiding behind a misleading single-verb name.
